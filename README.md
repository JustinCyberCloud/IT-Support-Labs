## üìö Lab Projects

### Lab 1: IT Helpdesk Ticketing System
**Technology:** Python  
**Description:** Full-featured ticketing system with priority management, ticket assignment, status tracking, and resolution documentation.

**Key Features:**
- Create, view, and manage support tickets
- Priority levels (Low, Medium, High, Critical)
- Ticket assignment and status updates
- Add notes and resolutions
- Persistent storage with JSON

**Skills:** Ticket lifecycle management, data persistence, user interface design

---

### Lab 2: Password Reset Automation Script
**Technology:** PowerShell  
**Description:** Automated password management tool with security validation and audit logging.

**Key Features:**
- Generate secure random passwords
- Validate password strength and complexity
- Simulate user password resets
- Activity logging for compliance
- Export logs to CSV

**Skills:** PowerShell scripting, security best practices, audit logging, automation

---

### Lab 3: Network Troubleshooting Toolkit
**Technology:** Python  
**Description:** Command-line network diagnostics tool for troubleshooting connectivity issues.

**Key Features:**
- Ping hosts for connectivity testing
- Traceroute to identify network paths
- DNS lookup (forward and reverse)
- Port scanning for service verification
- Generate comprehensive network reports
- Batch testing multiple hosts

**Skills:** Network fundamentals (TCP/IP, DNS), diagnostic methodology, reporting

---

### Lab 4: Active Directory User Management
**Technology:** PowerShell  
**Description:** Simulated Active Directory user management system for employee account operations.

**Key Features:**
- Create single or bulk user accounts
- Modify user properties
- Enable/disable accounts
- Group membership management
- User reports and search functionality
- CSV import/export

**Skills:** Active Directory concepts, user lifecycle, bulk operations, PowerShell automation

---

### Lab 5: System Health Monitoring Dashboard
**Technology:** Python  
**Description:** Real-time system monitoring tool with threshold alerts and performance tracking.

**Key Features:**
- Monitor CPU, RAM, disk, and network usage
- Customizable threshold alerts
- View running processes by resource usage
- Check service/process status
- Generate health reports
- Historical metric logging

**Skills:** System administration, performance monitoring, alerting, process management

---

### Lab 6: Remote Desktop Connection Manager
**Technology:** Python  
**Description:** RDP connection profile manager for quick access to multiple remote systems.

**Key Features:**
- Save and organize RDP connection profiles
- Quick connect to saved servers
- Search and filter connections
- Connection history tracking
- Test server availability
- Export/import connection profiles

**Skills:** Remote access management, data organization, user experience design

---

### Lab 7: Automated Onboarding/Offboarding System
**Technology:** Python  
**Description:** Complete employee lifecycle management system simulating IT onboarding and offboarding workflows.

**Key Features:**
- New employee onboarding workflow
- Account creation across multiple systems
- Asset assignment and tracking
- Access provisioning
- Employee offboarding process
- Asset recovery tracking
- Complete audit trail

**Skills:** Employee lifecycle management, workflow automation, asset tracking, compliance

---

### Lab 8: Knowledge Base with Search
**Technology:** Python  
**Description:** IT support knowledge base system for documenting and searching solutions to common issues.

**Key Features:**
- Create and manage knowledge articles
- Category organization
- Full-text search functionality
- Article rating system
- View statistics and popular articles
- Export/import knowledge base

**Skills:** Knowledge management, search algorithms, content organization, documentation
