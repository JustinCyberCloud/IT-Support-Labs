# IT Support Labs Portfolio

Hands-on IT support labs designed to simulate real-world help desk and desktop support tasks.

This repository showcases practical experience with common IT support responsibilities such as ticketing workflows, password resets, Active Directory user management, network troubleshooting, system monitoring, and basic automation.

These labs are intended to demonstrate job-ready skills for entry-level IT Support, Help Desk, and Desktop Support roles.

## Lab Projects Overview

- **Lab 1: IT Help Desk Ticketing System** (Python)  
  Simulates ticket creation, prioritization, assignment, and resolution workflows.

- **Lab 2: Password Reset Automation** (PowerShell)  
  Automates password resets with validation and audit logging.

- **Lab 3: Network Troubleshooting Toolkit** (Python)  
  Command-line tools for ping, traceroute, DNS lookup, and port scanning.

- **Lab 4: Active Directory User Management** (PowerShell)  
  Simulates user account creation, modification, group management, and lifecycle tasks.

- **Lab 5: System Health Monitoring** (Python)  
  Monitors CPU, memory, disk usage, and generates alerts.

- **Lab 6: RDP Connection Manager** (Python)  
  Manages saved remote desktop connections and availability checks.

- **Lab 7: Automated Onboarding / Offboarding System** (Python)  
  Simulates employee lifecycle workflows including access provisioning and asset tracking.

- **Lab 8: Knowledge Base System with Search** (Python)  
  Documents IT solutions with searchable articles and usage tracking.

## Detailed Lab Documentation

Each lab includes its own directory with detailed documentation, code examples, and usage instructions.
